PROYECTOS

Gestion de Cookies SW

Desarrollo para la gestión de cookies en los sitios web de una compañia. El pop-up informa al visitante sobre el uso de cookies y ofrece opciones para aceptar o rechazarlas. Si se aceptan, se permite el acceso completo al sitio web, y si se rechazan, solo se permite acceder a las partes básicas. El pop-up se diseñó para ser fácil de usar y se probó en diferentes navegadores y dispositivos para garantizar su funcionamiento óptimo.

Migración Azure

Migracion de sitios web y aplicaciones desde un ambiente en On-premise a la nube utilizando Docker y Kubernetes. El proceso fue gradual y cada sitio web y aplicación se migró individualmente y se probó antes de su puesta en producción. La migración ha permitido una mayor escalabilidad, flexibilidad y seguridad, y ha mejorado significativamente el rendimiento de los sitios web y aplicaciones gracias a Kubernetes que escala automáticamente el número de contenedores que manejan el tráfico.

Migración UA - GA4

Cambio de (UA) a Google Analytics 4 (GA4) a través de Tag Manager en 320 sitios y aplicaciones web. Se configuraron metas y objetivos de UA y se realizaron pruebas de calidad antes de la implementación final. Para que los administradores tengan acceso a información valiosa para mejorar el éxito de las campañas de marketing en línea.

Dockerizacion de un E-comerce
